# Dimetrodon
A 36 key split keyboard with stagger and splay, built with Ergogen.

### Images
![top](ccblaisdell.github.io/dimetrodon/top.png)
![bottom](ccblaisdell.github.io/dimetrodon/bottom.png)

## Development

- Clone the git repository.
- Install NodeJS.
- Run `npm install`.
- Run `make build`.
